・  Configure File Structure

In console mode configure file, general setting and command setting are included.

All the values in the configure file comes from the configuration in Flash Tool UI, and the values could be customized with correct format if necessary.

 

・   General Setting

・         chip-name: specific target chip name, such as MT6575, MT6577…

・         storage-type: specific the flash type, such as NAND, EMMC…

・         download-agent: specific the Download Agent file path.

・         scatter: specific the scatter file path.

・         authentication: specific the authentication file path.

・         certification: specific the certification file path.

・         rom-list: specific the path of images expected to be downloaded on the target.

・         connection: specific the connection relative configure.

–     USB

type: BromUSB

high-speed: configure download speed, if true, download will be in high speed;  if false, download will be in full speed when BootRom Download Mode.

power: configure target power supply, if WithoutBattery, power supply comes from USB, and battery is optional; if WithBattery, power supply comes from battery and battery is required; if AutoDetect, power supply from battery or USB VBUS is auto detected by DA.

da_log_level: configure tracing log level. Trace, Debug, Info, Warning, Error, Fatal are available.

timeout-count: configure connection timeout limitations.

com-port: configure which com port will be used when scan.

–     UART

type: BromUART

port: specific the correct com port number, like COM1…

・         checksum-level: specific the checksum level. if usb, the USB checksum will be performed; if storage, the storage checksum will be performed; if both, it means the USB checksum and storage checksum.

・         log-info: console mode log switch and time setting to delete log file regularly, it is generated by ui.

log_on: console mode log switch, enable log if true, otherwise false.

log_path: the directory in which the log files has been stored.

clean_hours: the time setting to delete log files regularly, the unit is hours.

 

 

・  Command Setting

Command Setting includes basic configurations for the operation.

・     Format

–     Auto Format

validation: switch the format validation is on or off, true is on and false is off.

physical: specific the operation is physical format or logical format.

erase-flag: specific the erase flag, NormalErase, ForceErase and MarkBadBlock are available.

auto-format: true

auto-format-flag: specific auto format flag. ”FormatAll”, “FormatNVRAM” and “FormatExcept_BL” are available.

 

 

–     Manual Format

validation: switch the format validation is on or off, true is on and false is off.

physical: specific the operation is physical format or logical format.

erase-flag: specific the erase flag, NormalErase, ForceErase and MarkBadBlock are provided.

auto-format: false

begin-addr: specific the begin address of the format area.

length: specific the length of the format area.

part-id: specific the region of the format area.

 

 

・     Download

FlashTool can export three different download command setting according to specific download scene seleted.

The three download scene are:

–     Download Only

da-download-all: Dummy command setting, as there is no setting yet.

 

 

–    Format All + Download

combo-format: Specific format command setting, details same as format command setting section.

da-download-all: specific download command setting, details same as download only setting section.

 

 

–     Firmware Upgrade

scene: Specific download scene, always FIRMWARE_UPGRADE.

readback: specific readback command setting, details same as readback setting section.

format: Specific format command setting, details same as format command setting section.

da-download-all: specific download command setting, details same as download only setting section.

write-memory: specific write memorycommand settin.

 

 

 

・     Readback

is-physical-readback: specific the operation is physical readback or logical readback, if true, physical readback will be applied and if false, logical readback will be applied.

readback-list: list all the readback rom item information.

 

 

・  Configure File Generation

Please make sure all the configurations mentioned in “General Configuration” has been ready before generate the configure file.

 

・  Select Operation

Configure File for Console Mode could be generated by Flash Tool GUI.

Press “[Menu] File-> [Submenu] Export Console Mode XML” to select the operation expected.

Support Operations: Format, Download, Readback.

 

 

・  Save File

Save the configure file with specific file path and name.

Default path is SP Flash Tool path and default file name is [operation].xml.
