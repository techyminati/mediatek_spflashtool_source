There are two ways to use console mode.  Each way provides different mandatory and optional arguments for different aims.

 

，  Use config.xml

–     Mandatory Auguments :

，     -i, --config: Specify console mode configuration file, which export from GUI.

Usage :

flash_tool -i config.xml

 

–     Optinal Auguments :

，     -p, --com_port : Specify com port, format is as COM4+COM6 (BROM download), COM5 (preloader download) on Windows and format is as 5-2+1-6(BROM download), 5-1(preloader download) on Linux.

Usage :

flash_tool -p COM4+COM6 -i config.xml

flash_tool -p COM4+COM6 -s MT6575_Anroid_scatter.txt -c download

 

，     -h, --help : Display help infomation and exit.

Usage :

flash_tool -h

 

，     -r, --redirect : Redirect debug log to parent process.

Usage :

flash_tool -r -i config.xml

 

，     -o, --efuse_read_only :Efuse read only, ONLY on Windows .

Usage :

flash_tool -o -i config.xml

 

，     --disable_storage_life_cycle_check :disable storage life cycle check feature.

Usage :

flash_tool --disable_storage_life_cycle_check -i config.xml

 

，     -b, --reboot : Reboot the device .

Usage :

flash_tool -b -i config.xml

 

，  Not use config.xml

–     Mandatory Auguments :

，     -s, --scatter : Specify scatter file path.

Usage :

flash_tool -c download -s MT6575_Anroid_scatter.txt

 

，     -c, --operation : Specify flash tool features, just support:

format\download\format-download\firmware-upgrade\cert-download\efuse\dram_repair.

Usage :

flash_tool -c download -s MT6575_Anroid_scatter.txt

 

 

–     Optinal Auguments :

，     -p, --com_port : Specify com port, format is as COM4+COM6 (BROM download), COM5 (preloader download) on Windows and format is as 5-2+1-6(BROM download), 5-1(preloader download) on Linux.

Usage :

flash_tool -p COM4+COM6 -i config.xml

flash_tool -p COM4+COM6 -s MT6575_Anroid_scatter.txt -c download

 

，     -d, --da_agent : Specify download agent file path.

Usage :

flash_tool -d MTK_AllInOne_DA.bin -s MT6575_Anroid_scatter.txt -c download

 

，     -e, --cert_file : Specify certification file path.

Usage :

flash_tool -e cert_file.cert -s MT6575_Anroid_scatter.txt -c cert-download

 

，     -h, --help : Display help infomation and exit.

Usage :

flash_tool -h

 

，     -r, --redirect : Redirect debug log to parent process.

Usage :

flash_tool -r -i config.xml

 

，     -t, --battery_mode : Specify battery mode. with, without and auto are available. if no specify the -t argument, the auto mode used.

Usage :

flash_tool -d MTK_AllInOne_DA.bin -s MT6575_Anroid_scatter.txt -c download -t without

 

，     --rsc : Specify the project name in rsc.xml, ONLY support download/format-download/firmware-upgrade for internal tool.

Usage :

flash_tool -s MT6575_Anroid_scatter.txt -c download --rsc k75v1_64_rsc_cn[op02]

 

，     --disable_storage_life_cycle_check :disable storage life cycle check feature.

Usage :

flash_tool --disable_storage_life_cycle_check -i config.xml

 

，     -b, --reboot : Reboot the device .

Usage :

flash_tool -b -d MTK_AllInOne_DA.bin -s MT6575_Anroid_scatter.txt -c download

 

NOTE: Please make sure that the operation and the configure file is under flash tool path. Or absolute path for flash tool and configure file is required.
